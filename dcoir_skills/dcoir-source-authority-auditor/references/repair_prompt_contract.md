# Paste-ready repair prompt contract

A repair prompt must include:
- the project name: AFRICOM_SOC_IR / DCOIR
- the current authority model: Airtable-first operational authority
- the exact drift symptom
- sources already checked
- the suspected stale file, skill, table, or workflow
- what not to do, such as creating duplicate Work Items or trusting old GitHub todo files
- requested output class: skill patch, Project attachment patch, GitHub Desktop bundle, Airtable update plan, or analysis only
