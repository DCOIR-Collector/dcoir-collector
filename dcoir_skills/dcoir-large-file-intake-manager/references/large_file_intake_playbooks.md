# Large-File Intake Playbooks

## too-large upload
- ask for metadata and file type first
- ask for the highest-value narrow excerpt next
- prefer summary plus one decisive excerpt over a broad dump

## missing file
- ask for the next best adjacent artifact
- pivot to metadata, event excerpts, command output, or screenshots if available

## partial upload
- state the bounded confidence limit
- analyze what is present
- request the most decision-relevant missing slice next
