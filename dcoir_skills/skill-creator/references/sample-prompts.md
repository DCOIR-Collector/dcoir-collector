# Sample prompts (copy/paste)

Use these to start a skill creation session. Keep them short and concrete.

```
Create a skill for weekly marketing updates. Ask 3 clarifying questions, then output a SKILL.md draft plus a short checklist for validating tone and length.
```

```
I want a skill that turns messy meeting notes into a structured action plan. Gather requirements, then produce SKILL.md and a simple template file for the output format.
```

```
Build a skill that summarizes legal documents for non-lawyers. Ask about jurisdiction and audience, then draft SKILL.md and a rubric for completeness and risk.
```
