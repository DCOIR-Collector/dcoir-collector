# Banner Message Templates

## Classes

### SESSION START
Use on the first substantive response in an AFRICOM_SOC_IR or DCOIR project chat session.
Examples:
- dcoir review in progress
- important dcoir architecture review in progress
- dcoir project work in progress

### MILESTONE
Use for major mid-conversation review points that should draw immediate attention.
Examples:
- important architecture decision ready for review
- major dcoir midpoint ready for review
- critical workflow boundary decision ready

### REVIEW
Use when the operator should inspect a result now but the response is not a direct gating question.
Examples:
- important dcoir result ready for review
- policy update candidate ready for review
- release-scope decision ready for review

### COMPLETE
Use when substantial work finished and the operator should notice the result.
Examples:
- requested dcoir task complete
- refresh bundle complete and ready for review
- deeper regression complete and artifacts are ready

### ACTION REQUIRED
Use when the operator must answer, approve, choose, or provide input.
Examples:
- operator approval required to proceed
- decision required on persistence choice
- operator answer required for next branch

### BLOCKED
Use when chatgpt cannot proceed without something material.
Examples:
- blocked pending authoritative source
- blocked pending operator approval
- blocked pending missing artifact or access
