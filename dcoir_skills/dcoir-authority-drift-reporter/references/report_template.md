# Authority drift report template

Required sections:
1. Triggering task
2. Observed symptom
3. Sources consulted
4. Expected current authority
5. Observed drift or conflict
6. Affected surfaces
7. Severity and drift family
8. Recommended repair lane
9. Safe next move
10. Paste-ready repair prompt

Keep reports compact enough to paste into a new session, but detailed enough that the next assistant can reproduce the issue without chat history.
