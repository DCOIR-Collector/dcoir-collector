# Read Me First

This bundle is the scanner-safer Gemini handoff packet for standing up the DCOIR Gemini agent set and giving the prime agent a usable reference corpus.

Open `Gemini_Build_Quick_Start.md.txt` first.

What this bundle includes:
- build files for the prime agent and four sub-agents
- the full current Knowledge doc set for prime-agent attachment and operator reference
- collector ZIP rebuild inventory and rebuild notes
- operator runtime notes explaining which runtime files were intentionally omitted from the bundle
- optional technical references that are not needed for normal setup

What this bundle does not include:
- the actual `DCOIR_Collector.zip`
- live script files for `DCOIR_Collector.ps1`, `run_DCOIR_Tests.ps1`, or `run_DCOIR_Tests.cmd`

Those scripts were intentionally omitted from the bundle. Their target runtime names and roles are explained in `03_OPERATOR_RUNTIME_NOTES/`, and their contents can be reconstructed later from the authoritative sources.
