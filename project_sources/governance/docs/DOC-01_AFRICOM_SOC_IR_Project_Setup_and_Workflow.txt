AFRICOM_SOC_IR Project Setup And Workflow

Purpose
- Describe the retained repository workflow at a high level.

Current model
- GitHub is the source of truth for repo content, workflows, tooling, and durable guidance.
- The compact `ircore` model governs routing, validation, reuse, and continuity behavior.
- Legacy Airtable-oriented material may remain for migration or verification only.

Change workflow
- use one scoped branch per coherent work item
- validate the affected surfaces
- open a reviewable PR
- merge only after readback confirms the intended state
