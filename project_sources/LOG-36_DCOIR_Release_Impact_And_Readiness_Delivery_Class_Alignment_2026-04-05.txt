DCOIR Release Impact And Readiness Delivery Class Alignment - 2026-04-05

Purpose
- Record the bounded delivery-class alignment pass across `dcoir-change-impact-analyzer`, `dcoir-release-scope-builder`, and `dcoir-promotion-readiness-reviewer`.

Decision summary
- Distinguish targeted skill updates, batched skill-update waves, GitHub Desktop manual repo-update bundles, and full-refresh project-upload bundles as separate delivery classes instead of collapsing them into a coarse targeted-versus-full-refresh model.
- Let `dcoir-change-impact-analyzer` escalate multiple changed skills to a batched skill-update wave and identify GitHub Desktop manual repo-update delivery as the normal next class for current governed repo-readable changes.
- Let `dcoir-release-scope-builder` classify GitHub Desktop manual repo-update bundles and batched skill-update waves directly instead of forcing them through older packaging labels.
- Let `dcoir-promotion-readiness-reviewer` block readiness when the settled delivery class is missing the required bundle, per-skill regression evidence, delivery instructions, or commit-summary guidance.

Why it matters
- The coordinated campaign now uses grouped GitHub Desktop repo updates and batched skill-install waves as normal operator delivery shapes.
- The previous release/impact/readiness family still reasoned too coarsely and could under-describe the real next delivery step or miss class-specific readiness gaps.
- Aligning these three skills together reduces drift between downstream work-list generation, packaging-class selection, and final readiness judgment.

Validation notes
- The updated `dcoir-change-impact-analyzer`, `dcoir-release-scope-builder`, and `dcoir-promotion-readiness-reviewer` skill packages pass clean packaging validation.
- Representative checks confirm that multi-skill change sets now surface batched skill-wave delivery, GitHub Desktop repo-readable changes now surface manual repo-update delivery, and readiness review now blocks class-specific missing prerequisites when the settled posture requires them.
