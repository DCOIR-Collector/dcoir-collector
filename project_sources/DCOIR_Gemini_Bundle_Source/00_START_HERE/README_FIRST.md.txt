# README FIRST

What this bundle is
- This is the operator-facing Gemini build bundle for the merged AFRICOM SOC Elastic alert triage and DCOIR workflow.
- It intentionally keeps the preferred folder structure and naming style from the older build bundle while replacing the old thin content with a fresh-generated parent agent and seven fresh-generated sub-agents.

What to use
1. Start in 00_START_HERE.
2. Read Gemini_Build_Quick_Start.md.txt.
3. Use the files in 01_GEMINI_AGENT_BUILD as the actual agent-definition content.
4. Use 02_PRIME_AGENT_ATTACHMENTS as the prime-agent attachment set.

What is not in this bundle
- No optional technical-reference folder.
- No repo control-plane files.
- No collector source files or harness.
- No extra rationale pack mixed into the agent-definition folder.
