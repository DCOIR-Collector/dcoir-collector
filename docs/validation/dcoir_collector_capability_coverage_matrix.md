# DCOIR Collector Capability Coverage Matrix

This source-maintained matrix is the human review companion for the harness-generated `collector_capability_coverage.json` and `collector_capability_coverage.md` artifacts. It exists to keep #187 from becoming an ad hoc pass/fail judgment: every advertised collector capability should either have strong FullRegression/manual-full-validation evidence, a deliberate synthetic-only boundary, a justified out-of-scope boundary, or a tracked gap.
