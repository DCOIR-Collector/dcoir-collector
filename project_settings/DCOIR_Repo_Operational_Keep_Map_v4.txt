DCOIR Repo Operational Keep Map

Purpose
- Provide high-level keep/delete guidance for active repository surfaces.

Keep
- source code, workflows, operator tools, durable docs, release history, and retained supporting assets that are still referenced by active repo behavior

Do not keep by default
- retired skill-mirror surfaces
- parity-refresh surfaces that exist only to maintain retired mirrors
- duplicated operating guidance that conflicts with the current retained model

Hard rule
- Do not delete a path solely because it looks old.
- Do delete or rewrite active-looking surfaces when they teach retired behavior or point to removed dependencies.
