# DCOIR Skill Memory

Purpose
- This folder stores helper-skill working-memory files for the AFRICOM_SOC_IR / DCOIR project.
- These files are human-readable helper state, not control-plane authority.
- They are intentionally separated from governed project files such as project_sources/, knowledge/, project_settings/, release_notes/, and supporting_assets/.

Rules
- Re-anchor to Project Instructions, then CP-01, then CP-02 before reading or writing any file here.
- Do not treat files in this folder as authoritative control-plane inputs.
- Use one clearly named subfolder per helper skill when practical.
- Keep files markdown-first and easy for a human to inspect.
- Promote anything that becomes durable project truth into the governed project files instead of leaving it only here.
